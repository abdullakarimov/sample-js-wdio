# Sample WebdriverIO + Cucumber tests

## Installation

Tests require [Node.js](https://nodejs.org/) to run.

Presuming the archive has been unpacked into /sample-js-wdio/ folder, run the following commands:
```
cd sample-js-wdio
npm install
```

## Running

To run the tests:

```npx wdio run ./wdio.conf.js```
